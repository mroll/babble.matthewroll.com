018 - Slack as Necessary Ingredient

This week at work is a return to the idea that slack is a necessary ingredient for outsized success. There are a lot of things I see around work that can be improved. And that it wouldn’t even be that hard to improve them. Just some focused and careful thought to research and design solutions, and then some straightforward implementation. I’m talking about like code review tools, static analysis, performance improvements in different areas of the data pipeline, development process improvements like automated deploys and faster builds, etc etc etc.
